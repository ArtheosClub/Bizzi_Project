Version: 1.0

Status: Foundational Specification

## 1. Governance Charter

Governance Framework определяет правила существования цифрового предприятия Art of Business.

## 2. Constitutional Principles

- GP-001 Process Over People
- GP-002 Explicit Ownership
- GP-003 Auditability
- GP-004 Human Sovereignty
- GP-005 Corporate Memory

## 3. Governance Layers

- Layer 0 — Human Layer
- Layer 1 — Strategic Governance
- Layer 2 — Enterprise Governance
- Layer 3 — Domain Governance
- Layer 4 — Execution

## 4. Governance Core

Полный список из 28 агентов и их зон ответственности вынесен в отдельный
документ — см. AGENT_REGISTRY.md. Ниже — ядро governance-слоя (Layer 1–2):

- AG001 CEO Agent — высшая исполнительная роль, финальное решение по стратегии
- AG002 Chief Orchestrator — маршрутизация задач между агентами, назначение исполнителей
- AG003 AI Auditor — аудит решений и compliance
- AG004 Business Analyst — анализ и поддержка стратегических решений
- AG005 Risk Manager — оценка и управление рисками
- AG026 Knowledge Manager — корпоративная память и база знаний
- AG028 Escalation & Audit Support Agent — поддержка AG003 в обработке эскалаций

Остальные 21 агент (AG006–AG025, AG027) владеют операционными доменами
(Marketing, Sales, Operations, Supply Chain и др.) и подчиняются ядру
через структуру Reports To в AGENT_REGISTRY.md.
