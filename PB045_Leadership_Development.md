# PB045 Leadership Development Program

Version: 1.0
Status: Active Playbook
Related Capability: C12 People Management
Related Functions: PEO-LDR-001, PEO-LDR-002
Owner Agent: AG023 Learning & Development Agent
Escalation Path: AG023 → AG021 HR Manager → AG001 CEO Agent (для программ
уровня C-level)

## Purpose

Playbook описывает разработку и проведение программ развития лидерских
навыков — как для преемников из PB044, так и для действующих руководителей
департаментов.

## Trigger Conditions

- Результат PB044 Talent Review & Succession Planning (индивидуальный план
  развития)
- Плановый ежегодный leadership development цикл для всех руководителей
  департаментов

## Stage 1 — Idea (Development Needs Assessment)

**Owner:** AG023 Learning & Development Agent
**Decision Level:** L2

Действия:
1. Собрать индивидуальные потребности развития (из PB044 или самооценки)
2. Определить общие пробелы в лидерских компетенциях по компании

## Stage 2 — Analysis (Program Design)

**Function:** PEO-LDR-001 Leadership Development Plan
**Owner:** AG023 Learning & Development Agent
**Decision Level:** L2

Действия:
1. Спроектировать программу (менторство, внешнее обучение, ротация,
   проектная работа)
2. Для каждого участника — определить конкретные измеримые цели развития

## Stage 3 — Risk Review

**Owner:** AG021 HR Manager
**Decision Level:** L2

Действия:
1. Оценить нагрузку на текущие обязанности участника во время программы
2. Проверить бюджет программы против HR-бюджета (FIN-BUD-002)

## Stage 4 — Decision (Program Approval)

**Owner:** AG021 HR Manager (AG001 CEO Agent — для программ, затрагивающих
C-level или требующих существенного бюджета)
**Decision Level:** L2 (L4 для C-level программ)

Критерии:
- Цели развития привязаны к конкретным будущим ролям (особенно для
  участников из succession plan)
- Бюджет утверждён

## Stage 5 — Execution (Program Delivery)

**Owner:** AG023 Learning & Development Agent
**Decision Level:** L1

Действия:
1. Провести программу согласно плану (обучение, менторство, ротации)
2. Отслеживать прогресс на регулярных контрольных точках

## Stage 6 — Audit

**Function:** PEO-LDR-002 Leadership Development Plan Review
**Owner:** AG003 AI Auditor
**Decision Level:** L2

Действия:
1. Оценить достижение целей развития по завершении программы
2. Для участников из succession plan — обновить их статус готовности
   (PEO-SUC-001)

## Stage 7 — Knowledge Capture

**Function:** KNW-LES-001 Lessons Learned Capture
**Owner:** AG053 Knowledge Curator
**Decision Level:** L1

Действия:
1. Зафиксировать эффективные форматы развития (что действительно ускоряет
   готовность к руководящей роли)
2. Обновить программу для следующего цикла

## KPIs

- Goal Achievement Rate (% участников, достигших целей развития)
- Promotion Rate после завершения программы
- Program NPS (удовлетворённость участников)
