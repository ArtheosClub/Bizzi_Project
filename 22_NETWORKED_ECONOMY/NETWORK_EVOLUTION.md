# 18. Autonomous Network Adaptation

Autonomous adaptation is allowed only for:

- low-risk routing changes;
- temporary capacity balancing;
- alert routing;
- reversible workflow optimization;
- pre-approved non-structural tuning.

Structural network changes require governance approval.

# 19. Human Oversight Model

- major platform redesign;
- governance redesign;
- high-risk participant changes;
